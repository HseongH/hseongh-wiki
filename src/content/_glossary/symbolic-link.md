# 심볼릭 링크 (symbolic link)

**한국어 정착어**: 심볼릭 링크

파일이나 디렉터리를 가리키는 특수 파일. 가리키는 대상의 경로만 저장하며, 대상이 사라지면 깨진 링크가 됩니다. pnpm 은 node_modules 안에서 패키지를 가리키기 위해 심볼릭 링크를 사용합니다. Windows 에서는 정션(junction)을 대신 사용합니다.
