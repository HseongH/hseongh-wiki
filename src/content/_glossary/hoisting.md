# 호이스팅 (hoisting)

**한국어 정착어**: 호이스팅

깊이 중첩된 패키지를 상위 디렉터리(보통 node_modules 의 루트)로 끌어올려 평탄화하는 것. npm 과 Yarn Classic 의 기본 동작. 결과적으로 package.json 에 명시되지 않은 의존성도 접근 가능해지는 부작용이 생깁니다.
